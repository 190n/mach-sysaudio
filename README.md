Truly cross-platform, low-level, audio IO in Zig.

## Features

Playback and recording with backends for:

* Linux
  * PulseAudio
  * PipeWire
  * Jack
  * ALSA
* Windows: WASAPI
* macOS/iOS: CoreAudio
* WebAssembly: WebAudio
